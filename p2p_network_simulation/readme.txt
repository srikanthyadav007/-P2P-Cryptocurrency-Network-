To compile the program : g++ p2p.cpp -o p2p
To run the program:
 ./p2p arg1 arg2
 Enter no.of nodes in the network


Here arg1 = Percentage of slow nodes
     arg2= Percentage of Low CPU nodes


 After completion of program
  1) The block chain of each node will be saved in output.csv
  2) The network details will be save in ex.csv
